# 介绍

[@xiangheng08/qrcode](https://github.com/xiangheng08/xh-qrcode) 是一个基于 Web Components 的轻量级二维码生成器库。它使用 [Lit](https://lit.dev/) 和 TypeScript 构建，提供了多种类型的二维码组件，包括基础二维码、个人名片二维码和群二维码。

## 特性

- **Web Components 标准**: 基于 Web Components 标准构建，可以在任何现代浏览器中使用，无需额外的框架依赖
- **多种二维码类型**: 提供基础二维码、个人名片二维码、群二维码等多种类型
- **高度可定制**: 支持自定义颜色、形状、大小、logo 等属性
- **响应式设计**: 自动适配不同屏幕尺寸
- **TypeScript 支持**: 完整的 TypeScript 类型定义
- **轻量级**: 体积小巧，不会增加项目的负担

## 浏览器兼容性

由于使用了 Web Components 技术，需要现代浏览器支持：

- Chrome 60+
- Firefox 63+
- Safari 10.1+
- Edge 79+

对于较老的浏览器，可能需要引入相关的 polyfill。
